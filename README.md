# 📊 Student Score Analyzer

A Python data analysis project that processes student performance data from a CSV file and generates insights and visualizations using pandas and matplotlib.

## 🎯 Purpose
To demonstrate foundational data analysis skills including data cleaning, transformation, and visualization.

## ⚙️ Features
- Loads structured data from CSV
- Computes statistical insights (mean, min, max)
- Assigns grade categories (A–F)
- Calculates pass/fail distribution
- Visualizes:
  - Student score comparison
  - Grade distribution

## 🛠️ Tech Stack
- Python
- pandas
- matplotlib

## 📈 Key Skills Demonstrated
- Data analysis workflow
- Data cleaning and preprocessing
- Functional programming structure
- Data visualization
