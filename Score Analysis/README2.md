# 📊 Student Score Analyzer (Pandas Version)

## Description
This project is a Python-based data analysis tool that reads student scores from a CSV file and generates insights and visualizations. It uses pandas for data handling and matplotlib for data visualization.

## Features
- Reads structured data from a CSV file
- Calculates:
  - Average score
  - Highest and lowest scores
- Assigns grades (A–F)
- Computes:
  - Pass and fail counts
  - Grade distribution
- Generates visualizations:
  - Bar chart of student scores
  - Grade distribution chart

##  Technologies Used
- Python
- pandas
- matplotlib

## ▶️ How to Run
1. Install dependencies:
pip install pandas matplotlib
2. Ensure `scores.csv` is in the same folder as the script
3. Run:
score_analyzer_pd.py

- Student scores bar chart
