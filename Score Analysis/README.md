# Student Score Analyzer

## 📊 Project Description
This is a simple Python project that reads student scores from a file and provides statistical insights such as average score, pass/fail count, and grade classification.

## 🚀 Features
- Reads scores from a text file
- Calculates average, highest, and lowest performance indicators
- Assigns grades (A–F) based on score ranges
- Provides summary insights (pass/fail count, performance evaluation)

## 🧠 What I Learned
- File handling in Python
- Functions and modular programming
- Basic data analysis logic
- Conditional statements and loops

## ▶️ How to Run
1. Make sure Python is installed
2. Open `score_analyzer.py`
3. Run the program
4. Ensure `scores.txt` is in the same folder
